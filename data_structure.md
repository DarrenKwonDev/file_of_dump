## 자구

자구 : 동일한 자료형을 여럿 저장하는 구조

효율성 : 시간 복잡도와 공간 복잡도. 일반적으로 '최악의 복잡도'를 말함.

자구 선택 기준 :

1. 데이터에 접근하는 빈도
2. 데이터에 접근하는 방법(삽입, 삭제, 검색, 읽기[접근])

## 자구 종류

배열, 스택, 링큐, 덱,  
linked list  
hash table  
tree  
graph
